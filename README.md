# Simple-Node-Server
Simple server to use on Celes Front-End Challenge

Deployed on: https://simple-challenge-server.onrender.com

### To use in develop mode
- Create a `.env` file with:
  ```
  CELES_API=https://***/products.json
  API_KEY=shpat_***
  ```
- Run
  ```
  $ npm i
  $ node app.js
  ```
Now your server listening on por 6060 by dfault

**Note: You can change PORT adding PORT=#### on `.env` file**
